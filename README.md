The Custom Fonts plugin allows you to upload your own font files (.ttf, .otf, etc), and use them in user specified code blocks. This differs from other font plugins which add global fonts. This plugin is made specifically for myself and other conlangers who need an easy way to use their own custom writing systems in their conlang and/or writing system documentation.

The plugin isn't on the community plugins tab yet so to install for now:
- download the zip folder
- extract it in YourVault/.obsidian/plugins (may need to click view > show hidden files)
- enable the plugin in obsidian
